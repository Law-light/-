# 数据分析及可视化项目
数据来源： https://tianchi.aliyun.com/dataset/649
项目参考： https://blog.csdn.net/FontThrone/article/details/108128527
## python版本=3.11
使用Pandas, NumPy, Scikit-learn (KNN), Matplotlib, Seaborn, PyEcharts等库
基于淘宝用户行为数据，利用Pandas和NumPy构建了R（最近消费时间）、F（消费频率）、M（消费金额）三大核心指标。

采用K最近邻算法，基于已有的用户价值标签对全体用户进行精准分类，实现了用户的自动化、数据化分群。

通过Seaborn与PyEcharts进行多维度可视化，清晰展示了“高价值用户”、“潜力用户”等不同群体的特征与占比，为差异化运营提供了直接依据。

该项目旨在通过机器学习方法，对淘宝用户进行精准的价值分群。通过构建RFM模型，并应用KNN分类算法，将用户划分为不同的价值群体，从而帮助业务方理解用户结构，并制定针对性的留存、促活与转化策略。


● 技术栈： Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, PyEcharts
● 在电商行业竞争激烈的背景下，为了深入了解淘宝用户的购物行为，提高用户体验和运营效率，需要对淘宝用户行为数据进行分析。
● 基于超1亿条淘宝用户行为数据，运用python（Pandas，Numpy）进行数据清洗与处理，将晦涩的时间戳转换为具体时间分段分析处理
● 构建RFM模型对用户进行价值分群，通过监督学习的KNN分类算法识别出4个不同的核心价值群体（如高价值用户占比15%并提供了几乎60%的GMV，且该群体用户点击->收藏->购买转化率达50.93%可针对性维护用户体验）
● 使用Matplotlib/Seaborn和PyEcharts进行多维度可视化，并提供数据图表可交互性，精准展示用户群体特征与占比，为精细化运营提供数据支持。
● 通过KNN模型清晰揭示了用户价值结构，在后续数据测试集中具备良好泛化能力，可直接复用至后续用户分析场景并提前做出业务调整
